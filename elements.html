Global Layoffs: Data Cleaning and Analysis in SQL

This project involved end-to-end data cleaning and exploratory analysis of a global layoffs dataset using SQL. Raw data were staged, duplicates were removed using ROW_NUMBER(), text fields were standardized, dates were converted to proper formats, and missing values were handled through logical updates and self-joins. After preparation, EDA was performed to analyze layoffs by company, industry, country, stage, and time, including rolling totals and yearly company rankings using window functions. The project demonstrates strong SQL skills in data transformation, aggregation, and time-series analysis to generate meaningful workforce insights.

Student Dropout Prediction

This project analyzes questionnaire data from 450 students in Bangladeshi private universities to identify key factors influencing dropout risk. After data cleaning and preprocessing, multiple supervised machine learning models were trained for prediction. Model interpretability techniques, including feature importance, RFE, and SHAP, identified relationship satisfaction, adjustment difficulty, health condition, program satisfaction, and peer cooperation as key predictors, supporting data-driven early intervention strategies.
